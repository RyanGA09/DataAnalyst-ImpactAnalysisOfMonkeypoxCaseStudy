# Impact Analysis of Monkeypox Case Study

---

## Data Structure

```bash
ImpactAnalysisOfMonkeypoxCaseStudy/data
│
├── processed/                                                                            # Contains the processed data, used for EDA and visualization.
│   └── monkeypox_{start_year}_{start_month}_to_{end_year}_{end_month}_processed.csv      # Processed Data file
├── raw/                                                                                  # Raw dataset folder
│     ├── filtered                                                                        # Contains the filtered data, ready for analysis.
│     │   └── monkeypox_{start_year}_{start_month}_to_{end_year}_{end_month}_filtered.csv # Raw filtered data file
│     └── original/                                                                       # Contains the original, unprocessed data.
│         └── monkeypox_{start_year}_{start_month}_to_{end_year}_{end_month}.csv          # Raw original data file
└── README.md                                                                             # Dataset Documentation Information
```
